Overview

Harbour is the open/free software implementation of a cross-platform, multi-threading, object-oriented, scriptable programming language, backwards compatible with xBase languages. Harbour consists of a compiler and runtime libraries with multiple UI, database and I/O backends, its own build system and a collection of libraries and bindings for popular APIs. With Harbour, you can build apps running on GNU/Linux, Windows (32/64-bit), macOS, iOS, Android, *BSD, *nix, and more. Read more

Why use it?

  • Harbour is free software, 100% compatible with Clipper compilers.
  • Harbour is proven to be stable, robust and efficient.
  • Harbour is portable across multiple operating systems with the same code base.
  • You can use it to develop either open source or commercial applications. Read more
